corepost-preboot добавляет сетевой допуск к расшифровке на этапе initramfs.
Компонент читает локальный конфиг, поднимает сеть, запрашивает unlock token и открывает LUKS только при разрешении сервера.
Поддерживаются профили 2FA и 3FA с USB-фактором.
В репозитории есть hook, local-top script и утилиты для проверки в headless VM.
Все адреса, идентификаторы и таймауты задаются конфигурацией.
